# 麻雀互助校园

## 项目简介
`麻雀互助校园` 是一个校园互助平台，主要功能是让用户能够发布求助信息和提供帮助服务，促进校园内的互帮互助。

## 技术栈
- **后端**: `SpringBoot`
- **前端**:
    - `Vue.js` (网页后台管理端)
    - `Uniapp` (小程序用户端)

## 功能特性
- 📝 发布求助信息
- 💪 提供帮助服务
- 🤝 校园互助匹配
- 📱 小程序端用户交互
- 🖥️ 后台管理系统

## 项目结构
```
麻雀互助校园/
├── 后端服务 (SpringBoot)
├── 前端管理端 (Vue.js)
└── 小程序客户端 (Uniapp)
```
